## ğŸ’¾ Installation  

1. **Clone the repository**  
   ```sh
   git clone https://github.com/yourusername/mans-pack.git
   cd mans-pack
   ```

2. **Install dependencies**  
   ```sh
   npm install
   ```

3. **Run the development server**  
   ```sh
   npm run dev
   ```

4. **Build for production**  
   ```sh
   npm run build
   ```

5. **Preview the production build**  
   ```sh
   npm run preview
   ```
